Tecnologías Utilizadas

- Java 21

- Spring Boot 3.5.4 - Framework de aplicación

- React - Frontend

Instalación y Configuración

Clonar el repositorio 
git clone https://github.com/BitWizper/Deskpro-Tickets.git

Ejecutar el backend 
  cd sistema-de-Tickets 
  mvn spring-boot:run

Ejecutar el frontend 
  cd frontend 
   npm run dev
